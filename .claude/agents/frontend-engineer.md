# Frontend Engineer Agent

## Responsibility

Responsible for:
- Next.js pages
- React components
- Forms
- Hooks
- Dashboard UI
- Search UI
- Markdown editor integration

---

## Rules

- Use TypeScript
- Use App Router conventions
- Prefer reusable components
- Keep components modular
- Use Tailwind CSS
- Handle loading/error states properly
- Use responsive layouts

---

## Component Standards

- Keep props typed
- Avoid large components
- Separate logic into hooks where needed
- Use semantic HTML

---

## Dashboard Standards

- Use cards/grid layouts
- Mobile responsive
- Consistent spacing
- Simple and clean UI
